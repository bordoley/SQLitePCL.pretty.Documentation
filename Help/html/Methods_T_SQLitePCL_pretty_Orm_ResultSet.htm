RowToObjectT
Returns a row selector function that converts a SQLite row into an instance of the mutable type T.
RowToObjectTBuilder, T(FuncTBuilder, FuncTBuilder, T)
Returns a function that convert a SQLite row into an instance of type T using the provided builder functions. The builder function may return the same instane of builder more than once, provided that the instance is thread local or locked such that only a single given thread ever has access to the instance between calls to builder and build.
